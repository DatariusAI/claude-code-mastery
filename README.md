# Claude Code Mastery: AI-Augmented Software Engineering

> Central portfolio repository for the **Analytics Vidhya "Claude Code Mastery"** 7-week program — a structured curriculum covering AI-augmented software engineering with Anthropic's Claude Code.

## Program Overview

This program builds progressive expertise in using Claude Code as an AI pair-programming partner across the full software development lifecycle. Each week introduces new concepts, workflows, and engineering practices — culminating in a capstone project that integrates all skills.

The repository serves as the living portfolio: weekly artifacts, mini project submissions, documentation, and reference materials are organized here as evidence of applied learning.

---

## 7-Week Roadmap

| Week | Theme | Core Skills |
|:----:|-------|-------------|
| 1 | **Foundations & Claude Code Mental Models** | Claude Code setup, prompt fundamentals, codebase exploration, CLAUDE.md authoring, mental model development |
| 2 | **Prompt Engineering & Spec-Driven Development** | Advanced prompting patterns, specification-first workflows, feature implementation from specs, test-driven development with AI |
| 3 | **Workflow Automation & Safety** | Automated workflows, safety guardrails, pre-commit hooks, CI integration, responsible AI-assisted development |
| 4 | **Scaling in Large Codebases** | Navigating monorepos, large-scale refactoring, dependency management, architecture-aware prompting |
| 5 | **Multi-Agent Engineering & Organizational Tooling** | Multi-agent orchestration, custom tools, MCP servers, organizational workflow design |
| 6 | **CI/CD & Deployment Automation** | Pipeline design, automated testing, deployment strategies, infrastructure-as-code with AI assistance |
| 7 | **Debugging, Security, Governance & Capstone** | AI-assisted debugging, security auditing, governance frameworks, capstone integration project |
