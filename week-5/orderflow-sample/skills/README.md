# Skills Library — OrderFlow

This directory is where you will build your Skill library during the Week 5 mini project.

## How to Use This Directory

```
skills/
├── README.md               ← This file (Skill index)
├── SKILL_TEMPLATE.md       ← Copy this to start a new Skill
├── REVIEW_TEMPLATE.md      ← Use this to peer-review a partner's Skill
├── [your-skill-1]/
│   ├── SKILL.md
│   └── examples/
└── [your-skill-2]/
    ├── SKILL.md
    └── examples/
```

## Adding a New Skill

1. Copy `SKILL_TEMPLATE.md` into a new folder: `skills/[skill-name]/SKILL.md`
2. Fill in all 5 sections
3. Run 3 test cases and document results in the Tests table
4. Update the index table above
5. Ask a partner to review using `REVIEW_TEMPLATE.md`
